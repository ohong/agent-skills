# Agent Skills

A collection of skills for Claude Code.

## Skills

| Skill | Description |
|-------|-------------|
| [data-storytelling](data-storytelling/) | Data visualizations, infographics, charts, and interactive graphics in the style of NYT/FT editorial viz. D3.js, scrollytelling, choropleths, dashboards, and WebGL data viz. |
| [name-it](name-it/) | Structured naming methodology for brands, products, companies, projects, and creative endeavors. Generates, evaluates, and selects names through a 6-phase process. |
| [spec-writer](spec-writer/) | Converts research reports into implementation-ready technical specifications that a coding agent can build in a single pass. All technical decisions made autonomously. |
| [vibe-security-audit](vibe-security-audit/) | Audits and fixes security vulnerabilities in AI-assisted "vibe-coded" apps, especially those using Supabase, Firebase, or other BaaS platforms. Flags missing RLS, hardcoded secrets, and unauthenticated access. |

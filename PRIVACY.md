# Privacy Policy

**Mission** is a Claude Code plugin that runs entirely within your local Claude Code session.

## Data Collection

This plugin does **not** collect, transmit, store, or process any user data. It has no servers, no analytics, no telemetry, and no network calls.

## Local Files

The plugin creates `.mission/` files (plan, progress, learnings) in your project directory. These files remain on your local machine and are never sent anywhere by the plugin.

## Third Parties

This plugin shares no data with third parties.

## Contact

For questions about this policy, open an issue at [github.com/ohong/agent-skills](https://github.com/ohong/agent-skills/issues).
